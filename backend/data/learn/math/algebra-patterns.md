## Patterns

A **pattern** repeats or grows by a **rule**. In Grade 5 you often see:

- **Growing patterns:** add the same amount each step (e.g. start at **3**, add **7** each time → **3, 10, 17, 24, …**)
- **Shrinking patterns:** subtract the same amount each step

The **6th term** means “the 6th number in the list,” counting the first term as #1.

## Tables and rules

A table can show **input → output**. If the rule is “multiply by 4 and add 1,” then input **5** → **5 × 4 + 1 = 21**.

## Variables (light intro)

A **variable** (like **n** or **x**) stands for an unknown number.  
**n − 4.6 = 12.8** means “what number minus 4.6 equals 12.8?”
